# Problem Dimensions

Choose one diagnostic lens before asking follow-up questions:

- **throughput** — delays, backlog, turnaround time, capacity (use for ops/lending/logistics)
- **growth** — customer acquisition, visibility, conversion
- **profitability** — margins, pricing, utilization, mix
- **workforce** — hiring, retention, workload, skills
- **retention** — churn, repeat business, renewals
- **efficiency** — waste, rework, coordination tax

Metric-first rule: identify which business metric the user is optimizing before choosing hypotheses.
