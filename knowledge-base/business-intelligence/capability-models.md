# Universal Capability Model

Nine cross-industry capabilities used by `capability_engine.py`. Specializations are inferred from user evidence only — never from industry templates.

| Capability | Scope |
|------------|--------|
| acquire_customers | Lead generation, marketing, demand |
| convert_customers | Sales, qualification, closing |
| deliver_value | Fulfillment, production, service delivery |
| retain_customers | Renewal, churn prevention, loyalty |
| collect_revenue | Billing, invoicing, pricing, collections |
| manage_risk | Compliance, underwriting, regulatory |
| manage_capacity | Scheduling, staffing, utilization, backlog |
| manage_information | Data, reporting, visibility, forecasting |
| make_decisions | Approvals, prioritization, planning |

Gap detection links outcomes (margin, churn, cash, backlog) to capability weaknesses. No operating-model templates are applied.
