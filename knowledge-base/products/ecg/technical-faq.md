# ECG Technical FAQ

## Summary

Common technical questions about ECG Document Intelligence.

## Content

**Q: What ECG file formats are supported?**
A: Scanned PDFs, images (JPEG/PNG), CSV, WFDB, and EDF waveform formats.

**Q: Does it support 12-lead ECG printouts?**
A: Yes — image preprocessing and OCR are tuned for standard 12-lead printouts and Holter summaries.

**Q: Can output integrate with FHIR or HL7?**
A: Normalized schema output is designed for EMR and FHIR-compatible downstream mapping.

**Q: Is HIPAA considered in the architecture?**
A: See compliance.md — processing supports compliance-aware deployment models.
