# Retify Integration Guide

## Summary

Supported retail data sources and output destinations for Retify pipelines.

## Content

**Ingestion formats:** CSV, Excel, PDF, images (OCR), JSON, SQL dumps, ERP logs, POS exports.

**Common integrations:** Snowflake, BigQuery, Databricks as data warehouse targets; Airflow and dbt for orchestration; Tableau and Power BI for visualization.

**Entity types:** SKU, store, customer, vendor, inventory, sales transactions.
