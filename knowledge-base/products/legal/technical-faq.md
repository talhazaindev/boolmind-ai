# Legal Data Fusion Technical FAQ

## Summary

Technical FAQ for legal data consolidation use cases.

## Content

**Q: What file formats are supported at ingest?**
A: CSV, JSON, and XLSX are primary supported formats.

**Q: What is a Golden Schema?**
A: A canonical field set proposed by the system per cluster, used to map and merge source columns into harmonized records.

**Q: Is human review required?**
A: Step 5 (Resolution) is human-in-the-loop for quality issues; automated paths exist for low-risk mappings.

**Q: How is GDPR handled?**
A: Deploy with appropriate data residency, access controls, and audit trails; see security.md.
