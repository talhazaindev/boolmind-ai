# Forecasting Engine Features

## Summary

Technical and functional capabilities of the Forecasting Engine platform.

## Content

- **Stack:** FastAPI backend, PostgreSQL for feature store and results, containerized ML workers.
- **Models:** Multiple time-series and ML models with automated model selection and backtesting.
- **Granularity:** Forecasts at SKU, store, region, and category hierarchy levels.
- **Weather integration:** External weather feeds linked to demand signals for inventory planning.
- **Promotion module:** Campaign tracking, uplift measurement, discount optimization.
- **Inventory module:** Stockout risk scores, reorder recommendations, safety stock suggestions.
- **Store segmentation:** Cluster stores by performance patterns; surface best-practice playbooks.
- **APIs:** REST endpoints for forecast retrieval, scenario runs, and dashboard integration.
- **Not included:** Fleet management, driver workload optimization, clinical document OCR, or legal dataset fusion—these require **custom solutions** or other Boolmind products.
